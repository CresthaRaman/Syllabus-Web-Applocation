📝 Model Question — ENCT 302

Answer ALL questions.

  1. Explain the concept of semantic web. 4 Marks
  2. Design an HTML form for student registration with the following: name, email, phone, program selection (dropdown), and a submit button. 4 Marks
  3. Explain DOM manipulation and event handling using JavaScript with suitable code examples. Demonstrate add/remove DOM nodes dynamically. A college wants to develop a SPA for course selection using React JS. 10 Marks
    1. Draw an architecture diagram for SPA using React (components, props, state, router).
    2. Explain the flow of data from a backend API to UI components using fetch() and state hooks.
    3. Compare React with Angular and Vue.
  4. List and explain any three features of Modern JavaScript with code snippets (ES6+). 6 Marks
  5. Explain MVC architecture with a labeled diagram. 4 Marks
  6. Write a Django flow for: 8 Marks
    1. URL routing
    2. View handling form data
    3. Template rendering
    4. Session storage
    Design the complete request to response cycle using a block diagram.
  7. Explain JSON versus XML, with examples, advantages, and data parsing workflow. 4 Marks
  8. Consider a hospital management system designing Microservices Architecture. Explain with an architectural diagram highlighting: 5 Marks
    1. API gateway
    2. Service communication
    3. Independent deployments
    4. Database per service strategy
  9. Explain SQL Injection, XSS, and CSRF with examples. 4 Marks
  10. Explain why CORS is needed in full-stack applications. 4 Marks
  11. Write short notes on: 3 + 4 Marks
    1. Continuous Integration and Continuous Delivery
    2. Progressive Web Apps
