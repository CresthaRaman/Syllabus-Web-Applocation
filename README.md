# WAP Micro Syllabus — ENCT 302

A clean, structured web view of the **Web Application Programming (ENCT 302)** micro syllabus with interactive features.

## Live Demo

**[https://syllabus-kappa.vercel.app](https://syllabus-kappa.vercel.app)**

## Features

- Structured display of all 6 units with topics, depth codes, and hour breakdowns
- Hover tooltips for abbreviations (e.g., MVC, API, ORM)
- Direct links from each subtopic to corresponding study materials on [bidursapkota.com.np](https://www.bidursapkota.com.np/blogs/web-application-programming-complete-guide)
- Clean, professional, and responsive UI

## Units Covered

| Unit | Title | Lecture Hrs |
|------|-------|-------------|
| 1 | Introduction to Web Technologies | 8 |
| 2 | Client-Side Web Programming | 10 |
| 3 | Server-Side Web Programming | 9 |
| 4 | Web Services & APIs | 7 |
| 5 | Web Application Security | 6 |
| 6 | Deployment & Modern Trends | 5 |
